New activites have been added to complete the interactivity folder :

  • Balance allows to work with different equivalencies in masses on a visual way.

  • Cadran allows to work the number of the day with the four basic operations.

  • Calcul generates automatically a set of operations to resolve.

  • Train allows to work on the notions of increasing and decreasing numbers.

  • OpenStreetMap is an alternative to the GoogleMap application with open ressources and without advertising.

The design and the help of the interactivity have been improved as well.
