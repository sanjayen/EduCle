The tool «Capture web content» is available from the toolbar in the integrated Web browser mode. It allows you to grab a full page or medias (sounds, videos, animations, flash).

You can, based on the medias and their effective availability, to do the following operations :

- Download in the library (to be used without connection)

- Download on the current page (to be used without connection)

- Add the object as a link in the library (will need an internet connection)

- Add a link on the current page (will need an internet connection).

N.B : depending the configuration of the sites, you will not always be able to choose between those 4 possibilities (some will then be “greyed” which means that they are not active).
