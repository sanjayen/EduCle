The “Documents” mode allows the export and import as file archives of the folders containing the presentation documents or their sub-folders.

To create and archive, it is necessary to group the documents in a folder.

Select the folder you desireto export as an archive format and click on the button “Export” by choosing the option “Export to Sankoré UBX format”.

The software then generates an archive (.ubx) that you will name and save in the desired place.

To import an archive containing several files (.ubx), click on the button “Import” of the “Documents” mode.

It is, for now, not possible to import by batch several .ubz files available on a drive or a USB key.
