Each object can now be moved forward or backward in the scene

When you select the object with the pointer tool, the edition frame shows two new buttons on the lower left side. The first icon from the left lets you move, step by step, the position of the object to the front (compared to the other objects) while the second icon lets you move the object to the back.

A long clic on the icon will send the object all the way to the back or bring it all the way to the front. A visual signal, in the form of an animated circle, will let you know when the modification to the object has been applied.
