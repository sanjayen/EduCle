Il est possible d'ajouter des liens hypermédias sur des objets contenant des images fixes ou du texte.

Un menu permet de choisir de placer :

  • un lien vers une page web (nécessite une connexion internet),

- un lien vers une autre page de la présentation,

  • un lien qui permet de jouer un son au clic sur l’objet.

Pour ajouter une action, il faut sélectionner l’option présente dans le menu du cadre de l'objet. Cliquez sur « OK » pour valider votre action.

Pour retirer l'action, il suffit de repasser par ce même menu.
