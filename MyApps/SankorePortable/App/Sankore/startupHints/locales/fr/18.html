Open-Sankoré n’est pas disponible en mode fenêtre. Le logiciel occupera toujours toute la surface de votre écran.

Cependant, il vous est possible de passer en mode “bureau” ou de “masquer” Open-Sankoré.

Le mode «bureau» est accessible en cliquant sur l’icône «Bureau» dans la barre des outils.

Pour masquer open-Sankoré vous pouvez utiliser la commande «Ctrl+H» sous Windows et Linux ou «Cmd+H» sous MacOS.

Vous pouvez également utiliser la commande «Ctrl+Tab» sous Windows ou Linux ou «Cmd+Tab» sous MacOS pour passer d’une application à l’autre pour réaliser des copier-coller par exemple.
