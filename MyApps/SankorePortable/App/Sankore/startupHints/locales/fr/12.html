La bibliothèque propose un dossier "Signets" dans lequel il est possible de stocker des signets Web (ou bookmarks).

Lorsque vous utiliser un glisser-déposer depuis la bibliothèque vers la page, un widget est alors directement ouvert avec le contenu de la page web.

Si vous cliquez sur l'icône du signet, une page s’ouvre alors dans le navigateur Web intégré d'Open-Sankoré.

Un clic l’icône "Signet" dans la barre d'outils en mode Web ajoute automatiquement un signet dans le dossier «Signets».
