La biblioteca contiene una cartella "Segnalibri" nella quale si possono memorizzare dei link web (o bookmark).

Se si usa un'azione trascina-rilascia dalla biblioteca verso la pagina, viene aperto automaticamente un widget con il contenuto della pagina web.

Cliccando sull'icona del link si apre una pagina nel navigatore Web integrato in Open-Sankoré

In modalità Web, un clic sull'icona "Segnalibro" nella barra strumenti aggiunge automaticamente un link nella cartella "Segnalibri".
