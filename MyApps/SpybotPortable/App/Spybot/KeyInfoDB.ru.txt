<b>Тип данныx:</b> <i>REG_SZ</i>, <b>Диапазон:</b> <i>Путь и Имя файла</i>, <b>Значение по умолчанию:</b> <i>(Нет никакого значения по умолчанию для этого поля ввода.)</i><br/>
<b>Описание:</b><br/>
Определяет альтернативный пользовательский интерфейс для Windows.<br/>
Этот параметр сохраняет настройки пользовательского интерфейса Групповой Политики. Политика группы добавляет этот параметр в реестр, когда Вы разрешаете политику. Если Вы запрещаете политику или устанавливаете ее неконфигурированной, Политика Группы удаляет этот параметр из реестра.<br/>
Программа Проводник (Explorer.еxе) создает знакомый интерфейс Windows, но Вы можете использовать этот парамерт, чтобы определить алтернативный интерфейс. Когда этот парамерт появляется в реестре, система запискает интерфейс, который Вы определяете вместо Explorer.exe.<br/>
Значение: (Не в реестре), Смысл: Explorer.exe используется как интерфейс.<br/>
Значение: Имя исполняемого файла, Смысл: Указанная программа используется в качестве интерфейса.<br/>
Значение этого параметра должно содержать имя файла интерфейсной программы, включая расширение .exe. Если файл программы интерфейса отсуствует в папке, определенной в переменной Пути окружения системы, праметр должен содержать полный путь к файлу.<br/>
<b>Метод изменения:</b><br/>
Для изменения значения этого параметра используйте Групповую Политику. This entry corresponds to the Custom user interface Group Policy (User Configuration\Administrative Templates\System).<br/>
Этот параметр соответствует пользовательскому интерфейсу Политики Группы (User Configuration\Administrative Templates\System).<br/>
<b>Совет:</b><br/>
Чтобы найти папки, обозначенные в переменной Пути окружения, в Панели Управления дважды щелкните по значку "Система", выберете вкладку "Дополнительно", нажмите кнопку "Переменные среды", и затем, в разделе "Системные переменные", щелкнуть на Path.<br/>
Для получения более детальной информации об особых настройкаx Политики Группы, см. Windows 2000 Resource Kit Group Policy Reference.<br/>
Для получения obwej информации о Политикe Группы, смотрите раздел помощи Windows.<br/>
Для просмотра таблицы связей политик с соответствующими записями реестра, обратитесь к Group Policy Registry Table.<br/>
