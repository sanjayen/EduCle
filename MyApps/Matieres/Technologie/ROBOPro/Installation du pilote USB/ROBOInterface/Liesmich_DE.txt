Dieser KeUsb-Treiber unterstützt:

  Windows 2000
  Windows XP (32 und 64 Bit)
  Windows Vista (32 und 64 Bit)
  Windows 7 (32 und 64 Bit)

Zur Installation rufen Sie das Setup.exe Programm auf. 
Dadurch werden bei Vista und Windows 7 die Treiber vorinstalliert. Wird dann ein 
unterstütztes Produkt angeschlossen, findet das System den passenden Treiber und 
nutzt diesen.

Bei XP erscheint der Hardware-Assistent. Hier wählen Sie zuerst "Nein, diesmal nicht" 
(Verbindung mit Windows Update) und anschließend "Software automatisch installieren".
Der Treiber besteht den Windows Logo Test nicht, daher müssen Sie bestätigen, dass Sie 
die Installation fortsetzen möchten.

Für Windows 98 / Me verwenden Sie den älteren 32bit Treiber aus dem Verzeichnis "Win98".
