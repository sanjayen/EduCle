Supported File Format

Pepakura Designer 3 supports several main file formats.
The file formats and loadable content supported by Pepakura Designer are as follows:

Supported File formats for importing 3D model

Software Extension Loadable content
HexaKing Super(LE) (Japanese Software) 6ks, 6kt Polygon, Texture UV coordinate, Material, Object Layer
Metasequoia mqo Polygon, Texture UV coordinate, Material, Object Layer
3D Studio 3ds Polygon, Texture UV coordinate, Material, Object Layer
Wavefront obj Polygon, Texture UV coordinate. Material, Object Layer
Lightwave lwo Polygon, Texture UV coordinate, Material, Object Layer
AutoCAD dxf Polygon, Object Layer, Color
* read "3DFACE" section
Binary STL stl Polygon, Color
Google Earth kml, kmz Polygon, Object Layer
* read "triangles" section

Pepakura Designer may not always load data properly due to your 3D software version or saving options.

Supported File formats for Saving Unfolded Pattern

In Pepakura Designer, you can save unfolded patterns as image files other than the original file format (.PDO).The file formats and saved content supported are as follows:

Software Extension Saved Content
Standard Image Software BMP Content shown in 2D Pattern Window is saved.
Adobe Illustrator, etc EPS Lines and faces (with color, without texture) of development are saved. If the edges' join numbers are shown, the join numbers are also saved.
Standard 2D CAD Software DXF Lines and faces (with color, without texture) of development are saved. According to the type, lines are segmented into different layers.
