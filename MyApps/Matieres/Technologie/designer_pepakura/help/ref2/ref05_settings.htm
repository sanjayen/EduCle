[Setting]
Texture Setting
(-> detail)		 Checks and changes the color and texture setting of the 3D model. If you select a texture from the settings list, the area around the faces to which the setting is applied is shown in yellow or white. Yellow surfaces have UV-coordinates, so the texture image is reflected. White surfaces have no UV-coordinates, so the texture can not be configured.
Print and Paper Setting Sets the paper size in the 2D pattern window and the other print configurations.
(Same as [File]-[Page and Print Settings].)
Other Setting Sets various configurations of Pepakura Designer.
- Line Style（detail）
- Color（detail）
- Accelerator（detail）
- Others（detail）
- Advance（detail）
Reset to Default Resets the user's settings to the initial state.
This comes into affect when you start Pepakura Designer next.
Split Window Both the 3D model window and the 2D pattern window are shown.
If you moved the central border line, the areas are adjusted to be divided equally.
Show 2DPatternWindow Only Shows 2D pattern window only.


Texture Setting
Material List
The list of textures and colors specified in the 3D model appears. If you select a name, the same named texture is shown in the texture display area (when textures are set), and at the same time, the corresponding faces are shown in the 3D model window.

Thumbnail Image
When a selected material has texture, the thumbnail of the texture is shown.

[Specify Texture Image]
Substitutes specified file for a new texture. Specify the target image file in the dialog. UV coordinates for textures have to be set.

[Delete Texture]
Resets the textures. The faces are shown in the material color from before the textures were applied.


Other Setting - Line Style Configuration
[Cut Line]
Specifies the style of the cut lines.
Switches between [Solid] and [None].

[Mountain Fold Line]
Specifies the style of the mountain fold lines.
Select either [Solid],[Dot] or [None].
You can edit the dot pattern by pressing the [Edit] button.

[Valley Fold Line]
Specifies the style of the valley lines.
Selects either [Solid],[Dot] or [None].
You can edit the dot pattern by pressing the [Edit] button.

[Draw white line under dot line]
Draws the background of dot lines with white. Set this option when the surface color is dark, line is invisible, etc.

[Hide almost flat folding lines]
Omits the drawing of lines that are almost flat. Use this option to hide needless folding lines for assembly.
The value of [Angle that is considered to be almost flat] is the threshold of the edge angle to erase, and the smaller the number is, the fewer edges get shown. You can specify between 90 degrees and 180 degrees.
If you want to specify to erase edges one by one, you can use [2DPatternWindow]-[Edit Mode]- [Set Edge Color] and select the [Del] button.
Edit dot/dash line style
Enter values in the boxes and set the patterns of the dot or dash line.


Other Setting - Color Configuration
[Mountain Fold Edge]
Sets the color of the Mountain fold edges in the 3D model window.
[Valley Fold Edge]
Sets the color of the Valley fold edges in the 3D model window.

[Single Sided Edge]
Sets the color of the Single sided edges (border edges) in the 3D model window.

[User Spedified Open Edges]
Sets the color of the Open edges specified by user in the 3D model window.

[Background of Model Window (Top)]
Sets the top color of the Background color of the 3D model window.

[Background of Model Window (Bottom)]
Sets the bottom color of the Background color of the 3D model window.

[Faces without material]
Sets the color of the Faces without materials.

[Edit Handles in 2D Window]
Sets the color of handles used in 2D window.
[Reset to Default]
Resets configuration of each color to the default state.


Other Setting - Accelerator
You can edit accelerators for changing edit mode.

[Command]
Commands that can set accelerator.

[Key][Modifier]
Specifies a key (A-Z) and the modifiers [Alt][Ctrl][Shift].


Other Setting - Other Configurations
[Beep when open edge is set]
When you specify an edge to open in the 3D model, it beeps.

[Reverse zoom direction]
Reverses the zoom in/zoom out operation by the mouse wheel action.

[Place edge ID inside of face]
Places edge ID inside of face when edge ID is displayed.

[Check existence of new version on internet when start]
Checks existence of new version on internet when start Pepakura Designer.

[Keep back up files (with sequence number) when save a file]
Keeps back up files (with sequence number) when you save (over write) a file.

[Keep center of mass when rotate part]
Keeps center of mass of a part between before and after you rotate it.

[Rotate 3D model when check the correspondence]
Rotates 3D model when check the correspondence between 2D pattern and 3D model.

[Grid Size]
Specifies the size of grid in mm.

[Amount of Overflow Paint]
Specifies the amount of overflow paint.

[Location of Pepakura Viewer]
Specifies the path of Pepakura Viewer that is launched when a locked file is opened.


Other Setting - Advance
[Smooth texture image]
Smooth texture image by using nearest neighbor algorithm.

[Use MIPMAP for print]
Use MIPMAP algorithm when print a development which contains large texture images.

[Join adjacent edges in 3D model when open]
Join adjacent edges in 3D model when a model is loaded. This is the same operation with [3D Model Window]/[Join Adjacent Isolated Edges].

[Join adjacent edges in 2D pattern automatically]
Join adjacent edges in 2D pattern after a model is unfolded.This is the same operation with [2D Pattern Window]/[Join Adjacent Edges].

[Fix flap shapes when old PDO file is opened]
All flaps has 45 degree angle when the file is saved by Pepakura Designer Ver.1. This option modifies those flaps so that they do not stick out when they are pasted to the opposite face.

[Connect outline edges when export BMP]
Draws outlines of parts by a connected single polylines. This affects when lines are thick.

[Paint flaps with neighboring face color instead of oberflow paint]
Paints flaps with neighboring face color instead of oberflow paint. This option rollbacks the new feature to the old feature same with Pepakura Designer 2.

[Default angle of flap corner]
Sets the default angle of flap corner.

[Maximum Undo History]
Specifies the maximum undo history stack size.
