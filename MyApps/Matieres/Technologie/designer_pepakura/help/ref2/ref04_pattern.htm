[2DPatternWindow]
Edit Mode/
Select and Move		 Selects and moves part, text and image objects.
You can select multiple objects at once by specifying an area by a mouse drag select.
You can also select multiple objects by clicking with [Shift] key held down.
Edit Mode/
Rotate Part		 Rotates the selected part. If you select this menu option, all vertexes of the development are shown in red. First, click a point in red that is at the center of parts to rotate. Then, drag to a point in red for rotation.
The rotation snaps to each 45 deg interval.
Edit Mode/
Join/Disjoin Face		 When you apply the disjoin operation, the selected edge becomes green. When you apply a join operation, the pointed edge becomes red.
If you click a green edge, the connected faces get separated.
If you click a red edge, the corresponding edge is pointed to with an arrow. The arrow's direction changes according to the mouse location. The arrow points to the face nearest to the mouse. If you click again, the corresponding face is stuck. If the original face and the corresponding face belong to different parts, the whole part moves. Otherwise, only a single face moves.
Edit Mode/
Set Edge Color
(-> detail)		 Sets the edge colors. You can also hide edges.
Edit Mode/
Edit Flap
(-> detail)		 Moves, adds and deletes flaps and changes the width.
Edit Mode/
Input Text
(-> detail)		 Lets you add text to the development at the place where you clicked.
Edit Mode/
Check Correspondence between 2D and 3D		 Shows a list of parts. You can see the correspondence between 2D pattern and the 3D model by clicking a face.
Insert Image Locates images to 2D pattern.
The image can be located under or above 2D pattern.
You can change width and height by dragging red circles on the corners.
Show Flaps Switches between showing & hiding flaps.
Show Edge ID Switches between showing & hiding edge IDs.
Edges with the same numbered edges should be pasted together when assembling. The size of the number can be changed from the [Configuration]-[Edge ID Configuration] window.
Show Page Number Switches between showing & hiding the page number.
The page number is shown at the bottom of the page in the center of the screen. (Printout is not affected.)
Show Part Name Shows name of parts. (Printout is not affected.)
Apply Transparency of Print Setting Applies the transparency set on [Print Setting] dialog to the display.
Join Adjacent Edges Joins the adjacent isolated edges in 2d pattern.
Recalculate Parts Layout
(-> detail)		 Recalculates parts layout and resizes the parts layout.
Flip Pattern Flips the pattern.
Change the Scale of Development/
Specify the Scale
(-> detail)		 Change the scale of development by specifying the scale.
Change the Scale of Development/
Scale Up by 10%		 Scale up the development by 10%.
Change the Scale of Development/
Scale Down by 10%		 Scale down the development by 10%.
Reset View Position Resets the view position and shows the development so as to fit all pages into development view window.
Show Grid Shows grid lines.
Snap to Grid Make parts, strings and images snaps to a nearest grid when they are moved.
Show Snap Line Shows snap lines. Make parts, strings and images snaps to the lines when they are moved.
Copy Development to Clipboard Transfers the development to the clipboard in BMP format. You can migrate data without using files, when you use the development in other applications.
Without reference to the number of pages, the bounding box of the pattern is copied.


Edit Mode / Set Edge Color
Sets the edge colors. You can select edges one by one by clicking or you can select multiple edges at once in an area by mouse drag.

[Select Target]
Specifies the target edge types.
It is convenient when specific multiple edges are selected by area.

[Paint]
Set selected color to the picked edge.

[Select Color/Hide]
Select a color from a color dialog by clicking a colored square.
By clicking the icon of pipet, you can pick a color from 2D pattern window.
By clicking the icon of eraser, you can hide edges.

[Hide Edges Almost Flat]
This option hides all edges that angles are larger than the specified value.


Edit Mode / Edit Flap

The states [Switch] is selected.


The states [Change Shape] is selected. 		Edit flaps. You can change positions, add, delete, change width. The modifications specified in this dialog are applied to the selected flaps or edges. You can select multiple flaps at once by draggin mouse.

[Switch]
Switch the position of flap by clicking. You can change the way of switching by [Detail] setting.

[Change Shape]
The shape of selected flaps are changed to the settings of this dialog. You can change the hight and angles.

[Apply to All Flaps]
Settings of this dialog are applied to all flaps.


Edit Mode / Input Text
Lets you add text to the development at the place where you clicked.
If you click on text, you can edit the text.

[Font name]
Specifies font name. When you want to distribute PDO data, please note that if the receiver does not have the same font, the text may not be displayed as you wish. In this case, the specified font is replaced by the standard font (MS Gothic).

[Size]
Specify font size in points.

[Color]
Specify text color.

[Space]
Specifies space between lines in mm.

[Text]
Enter the text to be displayed.


Recalculate Parts Layout
Development Layout Method

[Auto]
The size of the development is automatically adjusted to fit paper size.
This is same as when the [Auto] check box on toolbar is checked.

[Manual]
Specify the assembled size or scale relative to the values in model.

[Assembled Size]
Specifies the assembled size in cm.
When this value is changed, the value of the [Scale] also changes.

[Scale]
Specifies the scale relative to the values in the model file. If you want to merge multiple .pdo files to build a large model, specify the same value.
When this value is changed, the value of the [Assembled Size] also changes.

[Spacing of Parts Layout]
Specifies the gap between parts in mm. The size of a flap is not counted for gap calculation.

[Place Parts so avoiding Page Borders]
When a development stretches across multiple pages, this function aligns it not to cross the page borders. However, if there is a part which can't fit onto one page, this part will cross page border.


Change the Scale of Development
[Hight][Width][Depth]
Specify the size of assembled model by millimeters.

[Scale]
Specify the scale of assebled model against to the original data.

[Change Font Size of Text]
According to the change of the scale of development, the size of charactors are also changed.
