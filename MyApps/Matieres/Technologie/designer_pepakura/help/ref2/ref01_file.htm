[File]
Open
(-> detail)		 Opens a file.
This opens files of the PDO format (Pepakura Designer data format) and all other supported file types.
* You can also open a file by dragging and dropping it on Pepakura Designer's window.
If the extension is not PDO, 6kt or MQO, a wizard dialog for adjusting coordinates is shown after the file is loaded.
Save Saves the current file with its current name.
The extension is .pdo.
If the file hasn't been saved before, a Save As file dialog appears.
Save As Saves the current file with another name.
Choose a location to save the file to and enter a new file name.
The extension is '.pdo'.
Reload 3D Model This command is used when the source model is modified.
Pepakura Designer reloads the source model and updates the 2d pattern according to the modification. You can continue editing the 2d pattern even if the source model was changed.
Import PDO File Imports another PDO file and merges it with the current PDO file.
You have to take care for the following things.
If the current object is unfolded, the imported file also has to be unfolded. If the current object is not unfolded, the imported file has not to be unfolded seperately.
The size of the 3D model and its position depends on the size and the original position of it in the original 3D CG software. ( Pepakura Designer can not scale and relocate 3D models, so you have to adjust these in the 3D CG software.)
Configurations such as the line style and the margin size are used from the current PDO file.
Export/
Save As Locked Format
(-> detail)		 Saves the current file with Pepakura Viewer's original file format.
If you save a file in this format, its data cannot be edited in Pepakura Designer without a password; this prevents your published data from being altered.
You set the password for editing it later whilst saving the file.
Export/
Pepakura Design2 format		 Exports current file as the file of Pepakura Designer 2.
Export/
Single BMP File		 Exports whole development to a single bmp, jpeg, gif, png, tiff image.
Export/
BMP File Per Sheet		 Exports development to bmp, jpeg, gif, png, tiff images. One file contains one sheet. Serial number is automatically added to the end of the file name.
Export/Vector Format Exports development to eps, dxf, emf file. These are vector format. Only lines in the development are exported.
Export/
BMP File for Texture		 Exports whole development to a single bmp, jpeg, gif, png, tiff image. At the sametime, UV info of 3D model for textureing is reset. You can design on the exported image, then apply the image as a texture via [Setting]-[Texture Setting].
Export/
3D OBJ File with UV Info		 Exports an OBJ file with UV info. The UV is set as the development.
Property Sets the name of author and comments to current file.
Model Info
(-> detail)		 Information about current model, such as number of faces, number of objects, is displayed.
Print and Paper Setting
(-> detail)		 Set paper type and configurations for printing.
Print Prints a development.
For information on your printer's setup, see your printer's instruction manual.
Print Preview Lets you check on the screen what will be printed before you actually print it.
Printer Setup Let's you set up your printer.
For information on your printer's setup, see your printer's instruction manual.
Exit Exits Pepakura Designer.
If the current file is not saved, a window will appear asking if you wish to save the unfolding or not.


Open
A standard Window's file open dialog appears.
If the file type is not .pdo, .6kt and .mqo, the load wizard appears. The wizard is as follows.
[Step1]
Define the normal direction of the faces.
[Close] Exit this wizard.
[Step2]
Define front side of the model by selecting appropriate color button.
[Step3]
Define the bottom side of the model by selecting appropriate color button.
[Step4]
Specify whether or not to invert the model.
This wizard can be accessed whenever you need it by selecting [3DModelWindow]-[Change Coordinates].


Export/Save As Locked Format
[Set Password]
Set a password for unlock.

[No Password]
Does not set a password. If this option is selected, no one can edit this file.

[Allow Changing Texture Images]
This option allows users to change texture images.

[Show Author Name and Comment when Open]
When users open this file, the author name and the comments entered from [file]-[Property] are displayed.


Model Info
[Number of faces]
The total number of faces in the current 3D model.
( When [Hide fold lines almost flat] is set on [Line Style Configuration] dialog, the number  of faces may differ from what you see.)

[Number of vertices]
The total number of vertices in the current 3D model.

[Number of materials]
The total number of materials in the current 3D model.

[Number of objects]
The total number of objects in the current 3D model.
You can add objects by selecting [Import PDO file] from the [File] menu.

[UV info for texture]
Shows whether or not the UV information for textures exists.
Usually UV information exists in a model if it has textures, but some formats don't.
Pepakura Designer can apply textures only to models that have UV information.

[Number of texture images]
The total number of textures used in the current 3D model.


Print and Paper Setting
[Paper type]
Set paper type used in 2D pattern window.
Set the size of paper same with the size used by a printer.

[Size]
Lets you set the size of the paper for laying out the unfolded pattern that is shown in the 2DPatternWindow.
This value should be same with the size as set in the [Printer Setup] dialog.

[Orientation]
Specifies the orientation of paper, [Portrait] or [Landscape].
This setting should be same as the orientation on the [Printer Setup] dialog.

[Side Margin (mm)]
Sets paper's left and right margin. It is shown with a dotted line in the 2D development View Window.
If a part is out of the paper's bounds, a new page will be automatically created.

[Top/Bottom Margin (mm)]
Sets paper's top and bottom margin. It is shown with a dotted line in 2D development View Window.
If a part is out of the paper's bounds, a new page will be automatically created.

[Line Width]
Sets the width of lines to print (This doesn't affect the display). The value 1 is the finest width on the computer & printer. The real width depends on the ability of the printer.

[Print lines clearly (Vector print)]
Prints lines in vector format. When this is set, you can't set the transparency.

[Print lines smoothly (Bitmap print)]
Prints lines in bitmap style. When this is set, you can set the transparency.

[Transparency]
You can set transparency between 0% (deep) and 100% (invisible).

[Print page number]
Prints page numbers on page's bottom margin. If page margins are small, page numbers may not be printed.

[Print alignment marks for multiple pages]
Prints alignment marks (cross-hairs) for aligning multiple pages. The cross-hairs are printed on the page's four corners.

[Resolution (for bitmap print)]
The value specified here will be reflected in the resolution of bitmap print. Normally 'Middle' is a high enough value.
