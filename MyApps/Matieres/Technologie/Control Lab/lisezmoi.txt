Ce document contient un complément d'informations pour l'utilisation de la version 
Windows 9X de Control Lab.

=====================
Matériel nécessaire
=====================
Control Lab version 1.0 pour Windows 9X nécessite un PC de configuration minimum:

a) 486 DX2 66MHz
b) 8MB RAM
c) Affichage VGA (640x480 avec 256 couleurs)
d) 3 MB d'espace libre sur le disque dur pour l'installation
e) Un port série disponible. Un port de type 16550 ou équivalent est recommandé.
f) Une carte son compatible MIDI

==================
PC basse fréquence
==================
Avec un PC ancienne génération (486 DX2 66MHz), Il est recommandé d'utiliser Control Lab seul
pour obtenir le meilleur contrôle des maquettes. 

=================================================================
Différences entre Control Lab Windows 9X et la version DOS/MAC 
=================================================================
* Les commandes suivantes n'ont pas été reprises:
  DONNEOUTILS
  EFOUTILS
  LISTEOUTILS
  Nous vous conseillons d'utiliser en remplacement DONNEPROCS. Pour un complément d'information
  à ce sujet consultez l'aide de Control Lab.

  RAMENEFORME
  - Vous pouvez désormais copier et coller des images (BMP) depuis le presse papier.
 
  FLARGC	La première valeur est prise en compte.
  FMOTIFC 	Est sans effet.
  RECYCLE	Est sans effet. La mémoire est automatiquement gérée par Windows.
  
La commande LIERPROJET est ajoutée. Elle permet de relier des projets entre eux. Pour 
un complément d'information à ce sujet consultez l'aide de Control Lab.
 
* Toutes les primitives traitant des couleurs utilisent 3 paramètres, un pour chaque 
  composante : rouge, bleue et vert.

* Il n'est plus possible d'utiliser la virgule comme raccourci pour la commande DESIGNE.

* Capteurs, actionneurs et objets de page ne peuvent pas être désignés par un nom de primitives.
  Il est impossible d'utiliser des espaces dans le nom : des capteurs, des blocs de texte, 
  des curseurs et des procédures.

* Les fonctions Mathématiques n'ont plus besoin d'être séparées par des espaces avec 
  les données.

* Plusieurs pages (procèdures, projet, configuration) peuvent être ouvertes, réduites et 
  agrandies.
  L'utilisateur peut choisir d'afficher une ou plusieurs pages au démarrage à partir de la 
  commande Préférences du menu Fichier. 

* Le manuel de référence et le fascicule "Développer une application" livrés sur ce CDROM sont 
  ceux de la version MSDOS. Ils sont complétés par l'aide en ligne de la version WINDOWS.

* Les applications SERRE et BRAS sont inclues sur le CDROM. Le livre du maître associé 
  à ces applications est utilisable avec les versions MSDOS ou WINDOWS. 


* Il est impossible d'utiliser des applications développées avec la version MS-DOS de 
  Control Lab avec la version 9X. Cependant vos pouvez sauver vos pages de procédures 
  avec la primitive SAUVETEXTE et les récupérer dans Windows 9X avec RAMENETEXTE ou 
  un copier/coller depuis un traitement de texte.


* Le programme d'installation de Control Lab Windows 9X installe automatiquement 
  un raccourci sur le bureau et une rubrique Control Lab dans le menu Démarrer.

  Vous pouvez au choix effectuer un double-clic sur l'icône du bureau ou sélectionner 
  Control Lab dans le menu Démarrer pour lancer votre logiciel.
