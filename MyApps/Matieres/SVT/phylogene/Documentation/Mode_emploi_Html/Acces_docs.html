Accès aux documents

C'est une possibilité offerte pendant les activités : Classer, Établir relations de parenté au collège, Établir des relations de parenté au lycée

Activer le bouton

Lorsque le bouton Afficher les documents est enfoncé, un clic sur une case du tableau fait afficher le document correspondant.

On peut ainsi, dans certains cas s'affranchir de l'étape de construction du tableau, tout en conservant la possibilité de visualiser les espèces et les caractères étudiés.
