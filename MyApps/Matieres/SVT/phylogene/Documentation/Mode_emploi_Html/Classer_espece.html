Classer des espèces dans des groupes


Cette activité qui fait suite à l'activité de construction des boîtes (Classer)

Les boîtes ont été construites (ou chargées à partir d'un fichier) et validées.

  • Définir une nouvelle espèce :
    • Cliquer sur le bouton
    • puis donner le nom de la nouvelle espèce : 
  • Remplir le tableau avec les états de caractères de l'espèce
  • Lorsque le tableau est rempli, placer l'espèce dans un groupe représenté par une boîte
    • Faire apparaitre le nom de l'espèce en cliquant sur le bouton
    • Déplacer ce nom jusqu'à la bonne boîte avec la souris (bouton gauche enfoncé)
    •   et relâcher le bouton de la souris.
      Si c'est le bon groupe, le nom s'inscrit sous les autres.
